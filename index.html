How to Play:

The Simon Game is an electronic game of memory skill based on remembering sequences.

The game will light a random square pad, for example RED. Remember which pad lighted up and press that square to move to the next level.

The game will light another random square pad, for example YELLOW. Now you need to hit RED, YELLOW.

The game lights up BLUE, now hit RED, YELLOW, BLUE.

Only a pro can get beyond Level 10. Good luck!
